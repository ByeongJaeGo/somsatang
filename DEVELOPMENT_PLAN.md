# somsatang 개발 계획

생일자와 선물자가 서로 부담 없이 선물을 주고받는 서비스 **somsatang**의 개발 로드맵입니다.

### mvp 만 확실하게 제작하자!!!

## 생성할 기능
1. 친구 에게 링크 보내기와 버셀에 배포한 링크 집어넣기 복사할 수 있도록.
2. 문의 사항 기능 만들기.다른 사이트의 문의 사항창 처럼 다른 사람이 볼 수 있게.
3. 데이터 베이스 파이어베이스에 등록자 정보 입력되게 하기.
4. 현재 내 친구목록 
5. 이제 검색창에 '생일 선물 추천' '생일선물 뭐하지' 이런 문구에 내 사이트가 뜨도록 하는 방법 알기. 


로그인-> 내 생일까지 남은 날d- day 맨위에-> 내 친구목록 (있으면 현재 날짜에서 가장 가까운 친구 생일 날순으로 목록 뜨기, 없으면 아직 등록된 친구가 없다고 한다.) -> 로그인 하면 내가 받고싶은 선물 버튼이 있다-> 들어가면 3개 물건이랑 가격 있다. // 생알자인 친구있으면 이름 옆에 선물하기 버튼 누른다-> 생일자인 친구의 3개 물건 중 하나에 선물하기 누른다. (최대3 만원 ) (만약 충전된 금액이 없다면, 알림창이 뜨고 금액 충전기능이 뜬다.) -> 금액을 입력하고 선물완료하면 편지지 창이 뜨고 익명 편지를 작성할 수 있는 기능이 생긴다.-> 작성하고 보내기하면 완료되어싸고 알림창이 뜬다.

## 내 친구의 생일.
0. 오늘 생일인 사람, 있으면 뜨고 없으면, 없다는 문구표시.
1.현재 날짜에 가까운 사람 디데이로 뜨는 기능. 

## 생일자
1. 생일 등록하기.
2. 받고 싶은 물건 등록하기.
3. 

## 선물자
1. 돈 충전할 수 있게 기능
2. 생일자에게 선물하기 버튼 누르면 3개의 선물중 선물하고 싶은 아이템에 선물하기 클릭, 1~3만원 후원 가능.



## 개발 계획표

| 단계 | 기능 | 설명 | 상태 |
|------|------|------|------|
| 1 | 랜딩 / 사전 등록 | 이메일·생년월일 수집, 출시 알림 안내 | 진행 중 |
| 2 | 회원가입 / 로그인 | 이메일 기반 계정 생성 및 인증 | 예정 |
| 3 | 프로필 관리 | 닉네임, 생일, 프로필 공개 설정 | 예정 |
| 4 | 선물 위시리스트 | 받고 싶은 선물 3개 등록·수정 | 예정 |
| 5 | 선물 후원 | 선물자가 1만원 단위, 최대 3만원까지 후원 | 예정 |
| 6 | 링크 공유 | 생일자 페이지 URL 공유 및 미리보기 | 예정 |
| 7 | 출시 알림 발송 | 사전 등록자 대상 이메일 알림 | 예정 |
| 8 | 정식 출시 | 베타 종료, 전체 기능 오픈 | 예정 |

## 현재 범위 (베타)

- 프론트엔드 HTML 프로토타입
- 이메일 + 생년월일 입력 UI
- 출시 알림 안내 문구
- 실제 데이터 저장·이메일 발송은 미구현
